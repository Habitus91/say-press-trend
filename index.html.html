ESS8-1_PRESS_TREND 사용 매뉴얼
① 엑셀 데이터 입력 방법
항목 입력 방법
공정검사일YY.MM.DD 형식으로 입력 (예: 26.06.02)
Lot No.생산 Lot 번호 입력 (예: 260602)
주/야간드롭다운에서 주간 / 야간 선택
SPL No.샘플 번호 입력 (1, 2, 3 ...)
특이사항금형 수정 내용 등 특이사항 입력 (선택). 입력 시 대시보드 툴팁에 빨간색으로 표시됨
치수값각 치수 항목별 측정값 입력 (소수점 3~4자리)
⚠ CAP PLATE 시트와 VENT 시트 각각 입력. 날짜/Lot 셀은 병합되어 있으니 첫 행에만 입력하면 됩니다.
② 저장 및 대시보드 갱신 방법
1 엑셀에서 데이터 입력 완료 후 Ctrl+S 저장
2 저장 시 _config 폴더 안에 JSON 파일 2개 자동 생성
3 ESS8-1_PRESS_TREND.html 파일이 자동으로 갱신됨
4 열람자는 HTML 파일을 닫고 다시 열면 최신 데이터 확인 가능
✅ 파일 위치: 엑셀, HTML, _config 폴더가 모두 같은 폴더에 있어야 합니다.
③ 대시보드 보는 방법
[ 탭 전환 ]
상단의 CAP PLATE / VENT / 형합성(Cap-Vent) 탭을 클릭하여 아이템 전환
[ 정보창 ]
최신 측정일, 전체 데이터 수, 산포관리 기간, 3σ 이탈 건수 등 현황 요약
[ 산포관리 (좌측) ]
최근 15개 측정값 기준으로 3σ 관리
· 빨간 테두리 + ⚠ 뱃지: 3σ 이탈 발생 (즉시 확인 필요)
· 주황 테두리: 2σ 초과 (주의 관찰)
· 카드에 마우스 오버 시 → Trend 차트에서 이탈 위치 자동 표시
[ Trend 관리 (우측) ]
전체 기간 측정값 트렌드 표시 (도면 공차 기준)
· 빨간 점선: UCL/LCL (도면 공차)
· 초록 점선: 기준치
· 그래프 포인트에 마우스 오버 시 날짜, SPL, 측정값, 특이사항 표시
[ 형합성 (Cap-Vent) 탭 ]
CAP 벤트부 치수와 VENT 치수의 Gap을 분석
· Gap = CAP 벤트(가로/세로) 측정값 - VENT 가로/세로 평균값
· 스펙: 0 ~ 0.07mm / 범위 벗어나면 즉시 알람
④ 알람 기준
구분 기준 표시 조치
3σ 이탈최근 15개 중 1건 이상빨간 테두리 + ⚠ 뱃지 + 알람 바즉시 확인 필요
2σ 초과최근 15개 중 1건 이상주황 테두리만 (알람 없음)주의 관찰
Gap 이탈Gap < 0 또는 Gap > 0.07mm빨간 알람즉시 확인 필요
